# 冒泡排序

### 前言

**冒泡排序**算是计算机中最简单的一种排序了。其基本的思想就是交换，同为交换思想的**快速排序**是他的大哥。😂（废话不多说了，下面开始）



## 算法步骤

> 下面假设对数组进行从小到大的排序

### 最小操作

相邻两个元素进行比较。如果左边的元素大于右边的元素，则将左边的元素与右边的元素进行交换

### 

```c
//bubble sort
void BS(int a[])
{
    void display(int a[]);
    int i, j;
    printf("使用的是冒泡排序法：\n");
    for (i = 1; i < LEN; i++)
    {
        for (j = 1; j < LEN - i; j++)
        {
            if (a[j] > a[j + 1])
            {
                a[0] = a[j + 1];
                a[j + 1] = a[j];
                a[j] = a[0];
            }
        }
    }
}
```
